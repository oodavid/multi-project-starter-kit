# Setup Browsersync, SASS and Gulp

Sources

* https://browsersync.io/docs/gulp

```
npm init
npm install gulp gulp-sass browser-sync run-sequence del gulp-newer gulp-sass gulp-sourcemaps gulp-autoprefixer gulp-uglifycss --save-dev
```
